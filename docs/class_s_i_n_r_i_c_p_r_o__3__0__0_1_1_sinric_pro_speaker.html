Detailed Description

Device to control a smart speaker.

Supporting:

  • setVolume / adjustVolume
  • set / adjust / reset bands (BASS, MIDRANGE, TREBBLE)
  • mute / unmute
  • media controls:
    • FastForward
    • Next
    • Pause
    • Play
    • Previous
    • Rewind
    • StartOver
    • Stop
  • set mode (TV, MOVIE, ...)
